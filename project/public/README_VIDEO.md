# Hero Video Setup

## Video Location
Place your hero background video at: `project/public/videos/hero-video.mp4`

## Video Requirements
- **Format**: MP4 (H.264 codec recommended)
- **Resolution**: 1920x1080 (Full HD) or higher
- **Duration**: 30-60 seconds (looping)
- **File Size**: Optimize to under 10MB for web performance
- **Content**: Luxury private jet/aviation footage

## Recommended Sources
- Pexels Videos: https://www.pexels.com/videos/
- Pixabay Videos: https://pixabay.com/videos/
- Unsplash Videos: https://unsplash.com/videos

## Fallback
If no video is provided, the page will use a static background image as fallback.

## Example Video URLs (for testing)
The Home page includes a fallback video URL from Pexels. Replace with your own video file.
